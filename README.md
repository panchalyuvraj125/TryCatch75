# TryCatch75 🚀

A modern, dark-themed, glassmorphism web application to track college attendance, manage subjects, calculate bunking capacity, and view your weekly timetable. Designed for maximum efficiency and aesthetic appeal.

## Features ✨
- **Premium Glassmorphism UI**: A sleek dark mode interface with vibrant accents, cyber themes, and micro-animations.
- **Attendance Cockpit**: Dashboard to monitor overall attendance percentage, status (Safe/Danger), and daily learning streaks.
- **Subject Manager**: Add, edit, and track individual subjects with specific target attendance goals.
- **Bunk Calculator**: Simulate "What-If" scenarios. Instantly see how many classes you can afford to bunk or how many you need to attend to reach your target goal.
- **Weekly Timetable**: Visual grid of your classes with support for theory and practical batches.
- **Quick-Log System**: One-click logging of Present/Absent/Bunked status directly from the dashboard cards.
- **Local-First Storage**: Fast, serverless experience using browser `localStorage` with isolated multi-user support.
- **Command Palette**: Press `Ctrl+K` for quick navigation and rapid actions.
- **Onboarding Wizard**: Guided setup for new users to initialize their subjects and current attendance.

## Setup 🛠️
Just clone the repository and open `index.html` in your browser! No complex backend or build steps required.

```bash
git clone https://github.com/panchalyuvraj125/TryCatch75.git
cd TryCatch75
```
Open `index.html` in Chrome, Firefox, Safari, or Edge.

## Usage 💡
1. Register a new local account using the sign-up form.
2. Complete the onboarding wizard to initialize your subjects and starting attendance.
3. **Preset Timetable**: For the username `Yuvraj Panchal`, the Semester II CSE-AIML timetable (AM2 Batch) is automatically pre-loaded!
4. Use the sidebar to navigate between Dashboard, Calculator, Subjects, Timetable, and Profile settings.
5. Customize your avatar and UI theme in the Profile section.
